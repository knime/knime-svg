(Fragment came into existence in order to address https://knime-com.atlassian.net/browse/AP-12398)

This fragment contains unmodified source of Batik 1.8, which address an issue with Non-Oracle JREs
(com/sun/image/codec/jpeg/JPEGCodec got removed from non-Oracle Java but Batik 1.7- relies on it.)
